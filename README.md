# 🏆 종막컵(JONGMAK CUP) 경매 시스템 데이터 연동 가이드

이 문서는 경매 시스템에 참가자(선수) 명단을 불러오기 위한 구글 스프레드시트 작성법과 연동 방법을 안내합니다.

## 1. 스프레드시트 작성 양식 (필수)

새 구글 스프레드시트를 만들고, **첫 번째 줄(헤더)**에 반드시 아래 3가지 항목을 입력해야 합니다. 대소문자를 구분하지 않지만, 정확한 철자가 필요합니다.

### 📋 컬럼 규격

| 컬럼명 (Header) | 설명 | 필수 여부 | 예시 데이터 |
| :--- | :--- | :---: | :--- |
| **Name** | 선수의 닉네임 또는 이름 | ✅ 필수 | `Faker`, `Chovy` |
| **Position** | 선수의 포지션 (5개 라인) | ✅ 필수 | `TOP`, `JUG`, `MID`, `ADC`, `SUP` |
| **Tier** | 선수의 티어 정보 | ✅ 필수 | `Diamond 1`, `Challenger` |

> **⚠️ 주의사항**
> * **Position** 값은 반드시 영어로 입력해야 아이콘이 정상적으로 표시될 수 있습니다. (예: `TOP`, `JUG`...)
> * 빈 줄이 중간에 섞여 있으면 에러가 발생할 수 있습니다.

### 📝 작성 예시

| Name | Position | Tier |
| :--- | :--- | :--- |
| 제우스 | TOP | Challenger |
| 오너 | JUG | GrandMaster |
| 페이커 | MID | Challenger |
| 구마유시 | ADC | Master |
| 케리아 | SUP | Challenger |

## 2. 웹에 게시 (데이터 연동 준비)

작성한 시트를 시스템에서 읽어오려면 **'웹에 게시(CSV)'** 설정이 필요합니다. (단순 공유 링크로는 작동하지 않습니다!)

1. 구글 스프레드시트 메뉴에서 **[파일]** → **[공유]** → **[웹에 게시]**를 클릭합니다.
2. 팝업 창이 뜨면 다음 설정을 확인합니다:
   * **게시할 항목:** '전체 문서' 또는 해당 시트(예: '시트1') 선택
   * **형식:** '웹 페이지' 대신 **[쉼표로 구분된 값(.csv)]** 선택 (중요!)
3. **[게시]** 버튼을 클릭합니다.
4. 생성된 **링크(URL)를 복사**합니다.
   * 링크는 대략 이렇게 생겼습니다: `https://docs.google.com/spreadsheets/d/e/2PACX-.../pub?output=csv`

---

## 3. 시스템에 적용하기

1. 종막컵 경매 시스템 페이지에 접속합니다.
2. **'Google Sheet Link'** 입력창에 위에서 복사한 **CSV 링크**를 붙여넣습니다.
3. **[Load Data]** 버튼을 클릭합니다.
4. 선수 명단이 정상적으로 뜨면 팀장(Captain)을 선택하고 경매를 시작합니다.

## ❓ 자주 묻는 질문 (FAQ)

**Q. 데이터를 불러왔는데 아무것도 안 떠요.**
* A. '웹에 게시' 할 때 형식을 **PDF**나 **웹 페이지**가 아닌 **CSV**로 선택했는지 확인해주세요.

**Q. 포지션이 이상하게 나와요.**
* A. `Position` 컬럼에 오타가 있는지 확인해주세요. (TOP, JUG, MID, ADC, SUP 중 하나여야 합니다.)

**Q. 한글 이름도 되나요?**
* A. 네, `Name` 컬럼에는 한글, 영어 모두 사용 가능합니다.
