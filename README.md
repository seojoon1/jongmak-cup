# 🏆 2025 종막컵 (JONGMAK CUP) 경매 시스템

리그 오브 레전드 내전(자유랭크/스크림) 팀 구성을 위한 **실시간 경매 시스템**입니다.
구글 스프레드시트의 참가자 명단을 불러와 드래프트 경매를 진행할 수 있습니다.

---

## 📊 1. 구글 스프레드시트 작성 가이드 (필수)

경매를 진행하려면 참가자 명단이 적힌 구글 시트가 필요합니다.
**첫 번째 줄(헤더)**의 영문 철자를 정확하게 입력해주세요.

### 📋 컬럼(열) 규격

| 헤더명 (Header) | 설명 | 필수 | 예시 데이터 |
| :--- | :--- | :---: | :--- |
| **Name** | **호칭/닉네임** (방송용/디코용) | ✅ | `페이커`, `대상혁` |
| **IngameName** | **실제 롤 아이디** (소환사명) | ❌ | `Hide on bush` |
| **Position** | **포지션** (TOP, JUG, MID, ADC, SUP) | ✅ | `MID` |
| **Tier** | **티어 정보** | ❌ | `Challenger` |

> **💡 참고사항:**
> * `IngameName`을 비워두면 `Name`과 동일하게 표시됩니다.
> * `Position`은 반드시 **영어**로 입력해야 합니다.

### 📝 작성 예시

| Name | IngameName | Position | Tier |
| :--- | :--- | :--- | :--- |
| 페이커 | Hide on bush | MID | Challenger |
| 제우스 | Zeus | TOP | Challenger |
| 케리아 | Keria | SUP | Challenger |

---

## 🔗 2. 구글 시트 연동 방법 (웹 게시)

작성한 시트를 프로그램이 읽을 수 있도록 **CSV** 형태로 게시해야 합니다.

1. 구글 스프레드시트 상단 메뉴 **[파일]** → **[공유]** → **[웹에 게시]** 클릭
2. 팝업 창 설정 변경:
   * **게시할 항목:** '전체 문서' 또는 '시트1'
   * **형식:** '웹 페이지' ➡️ **[쉼표로 구분된 값(.csv)]** 으로 변경 (중요!)
3. **[게시]** 버튼 클릭
4. 생성된 **링크(URL) 복사**
   * 링크 끝이 `output=csv`로 끝나야 정상입니다.
